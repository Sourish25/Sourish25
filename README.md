## 🌾 Featured Projects

### [**Right Click to Harvest 🌾**](https://github.com/Sourish25/Right-Click-to-Harvest)
> Stop breaking your crops by accident! A lightweight, server-side only Fabric utility that enables instant right-click harvesting and smart replanting for standard crops, ancient Sniffer plants, cocoa beans, nether wart, melons, pumpkins, sugar cane, and cactus.

* **💻 Multi-Version Native Support**: Builds compiled specifically for Minecraft `1.20.x`, `1.21.x`, `26.1.x`, and `26.2-pre-2`.
* **⚙️ Zero Client-Side Installation**: Install on the server only, allowing vanilla clients to enjoy all features instantly.
* **🛡️ Durability Protection**: Break melons and pumpkins instantly with no durability cost!
