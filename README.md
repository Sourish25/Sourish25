### 📂 Featured Projects

#### 📱 [MineBucks 📈](https://github.com/Sourish25/MineBucks) — Creator Revenue Dashboard
* **Description**: A native Android application that aggregates, tracks, and analyzes creator earnings securely across Modrinth and CurseForge in real-time.
* **Architecture**: MVVM, Jetpack Compose, Kotlin Coroutines/Flow, Room DB, WorkManager, Jetpack Glance.
* **Key Feature**: Utilizes a secure, on-device parsing engine with WebKit User-Agent spoofing and custom DOM JavaScript injection to safely authenticate and extract stats locally, preserving absolute data privacy (`EncryptedSharedPreferences`).

#### 🤖 [OrgaNexus 🌌](https://github.com/Sourish25/OrgaNexus) — AI Logistics Orchestrator
* **Description**: An autonomous multi-agent swarm platform designed to schedule and manage large-scale corporate event logistics from textual drafts.
* **Architecture**: Next.js 14, FastAPI, LangGraph, Google Gemini API, Docker.
* **Key Feature**: Powered recursively via a LangGraph Directed Cyclic Graph (Swarm) structure with dynamic "Pulse Drift" rerouting and automated SMTP notification update loops.
